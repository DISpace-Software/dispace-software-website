# DISpace Software Website

Premium static website for DISpace Software with a flagship SeatMap product section, responsive layouts, animated device mockups, team profiles, and separate lightweight SeatMap beta/guide pages.

## Local Preview

```bash
python3 -m http.server 5173 --bind 127.0.0.1
```

Open:

```text
http://127.0.0.1:5173/
```

## Pages

- `index.html` — main company website
- `seatmap-beta.html` — SeatMap beta request page
- `seatmap-guide.html` — SeatMap product guide page
